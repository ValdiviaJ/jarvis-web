Funcionalidades del Asistente

Una suite integrada de automatización local, control inteligente y hardware integrado.

Control por Gestos de la Mano (OpenCV)

A través de tu cámara web, el sistema detecta la presencia de tu mano y procesa los defectos de convexidad en tiempo real. Configura gestos de 1 a 5 dedos para disparar acciones rápidas como abrir programas del sistema, activar la escucha de voz o borrar el chat.

Captura y Síntesis de Voz

Usa comandos hablados naturales en español procesados asíncronamente con threading. Jarvis captura tus solicitudes locales por micrófono y responde con síntesis TTS.

Inteligencia Artificial con Gemini

Un backend inteligente conectado a la API de Google. Mantiene memoria histórica contextualizada de tus últimas interacciones para chatear libremente.

Buscador Inteligente de Archivos

Realiza un barrido a profundidad 2 en carpetas clave (Documentos, Descargas, Escritorio, etc.), ordenando coincidencias por fecha de modificación para abrir de inmediato el archivo más reciente.

Arquitectura de Plugins Modulares

Todo comando de Jarvis es una clase que hereda de BaseCommand en commands.py. Esto permite agregar nuevos comandos y automatizaciones de forma totalmente limpia, simplemente definiendo palabras clave y una función lógica de ejecución.

Detección y Reconocimiento de Gestos con OpenCV

El procesamiento de imágenes de la cámara web se ejecuta a una frecuencia de refresco de ~30 FPS usando Tkinter y OpenCV. Se utiliza un algoritmo basado en filtrado morfológico y defectos de convexidad para contar los dedos levantados de forma offline y sin sobrecargar la CPU.

1. Filtro Espacial (ROI): Se extrae una Región de Interés en el lado derecho del visor (de un tamaño de 40% de ancho y 60% de alto de la resolución de cámara) para obligar al usuario a colocar la mano dentro de un área de detección estable.
2. Umbralización de Piel (HSV): Se convierte el frame a espacio de color HSV y se aplica una máscara de rango (H: [0, 20], S: [20, 255], V: [70, 255]) para aislar la silueta de la mano de forma robusta.
3. Defectos de Convexidad: Se calcula el contorno más grande y su envolvente convexa. Los puntos de mayor desviación entre ambos se identifican como "defectos". Si el ángulo de separación entre dedos es menor a 90 grados y la profundidad es estable, se detecta un dedo levantado.
4. Filtro de Estabilidad: Un comando solo se dispara si la cantidad de dedos se mantiene estable durante 15 frames consecutivos (~500ms) y tras superar un cooldown de ejecución de 2.5 segundos para evitar spam accidental.

Clasificador e Inteligencia de Respuestas

Jarvis procesa la entrada del usuario en base a tres niveles de prioridad dentro de chatbot.py:

Nivel 1
Modo Gemini IA (Online): Si el modo de Inteligencia Artificial está activo en el panel, la entrada se clasifica primero usando el modelo de Gemini. Se le provee una instrucción del sistema donde la IA decide entre ejecutar un comando nativo (ej. ABRIR_NOTEPAD, ABRIR_CALCULADORA) o conversar libremente con el usuario adoptando la personalidad de Jarvis.
Nivel 2
Enrutamiento Local Directo (Offline): Si no hay conexión o el Modo IA está desactivado, el comando pasa por expresiones regulares e inspección de palabras clave para buscar coincidencias lógicas simples como abrir el navegador, reproducir música o calcular operaciones matemáticas directamente mediante la API de Python.
Nivel 3
Fallback de Respaldo Local (Offline): Si la cuota API se agota (código HTTP 429), Jarvis redirige la consulta localmente a una búsqueda rápida en la enciclopedia de Wikipedia offline para resumir e informar al usuario usando TTS.

Algoritmo de Búsqueda de Archivos Locales

El comando /file <consulta> permite abrir archivos en el disco duro usando coincidencia difusa en directorios comunes del sistema (como Descargas, Documentos y Escritorio).

Para prevenir sobrecargas de rendimiento o tiempos de respuesta largos (típicos de búsquedas completas en disco), el algoritmo de búsqueda implementa las siguientes optimizaciones de rendimiento:

  • Profundidad Controlada: Utiliza el generador os.walk() limitando el recorrido del árbol de directorios a un nivel máximo de 2. Las ramas de subdirectorios que superan este nivel son purgadas en caliente durante el ciclo de búsqueda.
  • Ordenamiento por Modificación: Si se encuentran múltiples archivos que coinciden con el nombre buscado, la lista se ordena en base a la fecha de última edición del archivo (os.path.getmtime), seleccionando y ejecutando el documento más recientemente modificado por el usuario.

Arquitectura de Plugins Modulares

El proyecto sigue un patrón de diseño orientado a objetos para sus comandos, aislando la lógica de ejecución del asistente de los comandos en sí. Para crear un nuevo comando, basta con escribir una clase sencilla en commands.py: 

class OpenFacebookCommand(BaseCommand):
    @property
    def keywords(self) -> List[str]:
        # Palabras clave y comandos barra para activar
        return ["abrir facebook", "abre facebook", "/facebook"]

    @property
    def description(self) -> str:
        return "Abre Facebook en el navegador por defecto."

    def execute(self, text_input: str, app_instance=None) -> str:
        import webbrowser
        webbrowser.open("https://www.facebook.com")
        return "Abriendo Facebook, señor."

Posteriormente, en la función de inicialización del registro, agregas registry.register(OpenFacebookCommand()) y el nuevo comando estará disponible tanto por consola, entrada de voz, como por intents de Inteligencia Artificial.

Guía de Instalación Rápida

Configura y ejecuta J.A.R.V.I.S. en tu computadora localmente siguiendo estos sencillos comandos.

Paso 1

Clonar el código fuente

Clona el repositorio oficial de tu proyecto desde la terminal de Git.

git clone https://github.com/ValdiviaJ/jarvis.git
cd jarvis
Paso 2

Crear entorno virtual

Genera y activa el entorno virtual de desarrollo de Python para evitar conflictos.

python -m venv venv
venv\Scripts\activate
Paso 3

Instalar dependencias

Instala las librerías necesarias de visión por computadora y reconocimiento de voz.

pip install -r requirements.txt
Paso 4

Ejecutar el Asistente

Corre el archivo principal para inicializar la GUI y configurar tu API Key.

python main.py
