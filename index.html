Cómo Instalar en tu PC

Sigue estos sencillos pasos en tu consola para desplegar el asistente de forma local.

1

Clonar Repositorio

git clone https://github.com/tu-usuario/jarvis.git
2

Entorno Virtual

python -m venv venv
venv\Scripts\activate
3

Instalar Librerías

pip install -r requirements.txt
4

Ejecutar

python main.py

Lleva el Control a tu Computadora

Descarga la carpeta completa del proyecto. Incluye la interfaz gráfica con el reactor arc en Python, el control gestual y el nuevo asistente de inicio que configura tu API Key sin que toques una sola línea de código.
